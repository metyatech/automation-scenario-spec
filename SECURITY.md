# Security Policy

## Supported Versions

Only the latest version of the automation-scenario-spec is supported for
security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.0.0   | :white_check_mark: |
| < 2.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it to
the maintainers.

You can report vulnerabilities by:

1. Opening a GitHub Issue (if it is not sensitive).
2. Contacting metyatech directly if the vulnerability is sensitive.

We will acknowledge your report and provide an estimated timeframe for a fix.
