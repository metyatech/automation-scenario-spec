# Contributing to automation-scenario-spec

Thank you for your interest in contributing!

## How to Contribute

1.  Fork the repository.
2.  Create a new branch for your changes.
3.  Make your changes and ensure they follow the project's style and pass all tests.
4.  Submit a pull request.

## Development

Run the following commands to verify your changes:

```bash
npm install
npm run verify
```

## AGENTS.md

This project uses `AGENTS.md` for AI agent rules. If you update rules, please run `compose-agentsmd` to regenerate the file.
